# Despachos y Ventas App

Aplicación para consulta de despachos y ventas.  
Monorepo que contiene el frontend, dos backends (despachos y ventas) y la infraestructura como código.

## Requisitos previos

- **Java 17**
- **Maven/Gradle**
- **Node.js**
- **Docker** y **Docker Compose**
- **Terraform**
